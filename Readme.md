# clib-validate

  Validate a `package.json`.

## Installation

    $ clib install clib-validate

## Usage

```
$ clib help validate

  Usage: clib validate <dir>

    Validate a package.json in the given <dir>.

  Examples:

    # validate ./package.json
    $ clib validate

    # validate ./repos/foo/package.json
    $ clib validate repos/foo
```
